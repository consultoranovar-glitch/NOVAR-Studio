# Proyectos · trabajo por cliente

Un cliente/proyecto = una carpeta. Para partir uno nuevo, sigue `AGENTS.md` sección 4:

1. `marcas/<cliente>/design-tokens.json` (desde `marcas/_plantilla/`).
2. `proyectos/<cliente>/README.md` con: qué se produce, quién valida, decisiones tomadas,
   y qué versión de cada pieza es MÁSTER.
3. Producir con el motor, siempre parametrizado por los tokens.

El material de un cliente JAMÁS se mezcla con el de otro.
